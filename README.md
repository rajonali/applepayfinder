

# Apple Pay Finder

## Technologies
  React, MangoDB, Express, and Node

## Run Project

**Client:**

cd applepayfinder/

yarn add

**Server:**

mongod (need Mongodb installed)

cd backend/

yarn add\

yarn run dev



Description:
------------

Find Apple Pay accepting establishments from a user curated repository of verified locations.
